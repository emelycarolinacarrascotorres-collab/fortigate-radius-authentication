# fortigate-radius-authentication
Implementación de una infraestructura AAA centralizada mediante FreeRADIUS y FortiGate VM para autenticación RADIUS y control de acceso.

## 🏗 Arquitectura del Laboratorio

1. Ubuntu Server (FreeRADIUS)
2. FortiGate VM (RADIUS Client)
3. Usuarios Administrativos
4. Sistema de Autenticación Centralizada

## 📂 Estructura de la Documentación

- Fase 1: Preparación del entorno
- Fase 2: Instalación de FreeRADIUS
- Fase 3: Integración con FortiGate
- Fase 4: Pruebas de autenticación
- Troubleshooting
